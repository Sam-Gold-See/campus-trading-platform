# 复合搜索与筛选引擎 - 后端技术设计文档

## 1. 概述

支持关键词 FULLTEXT 搜索 + 多维结构化筛选（类型、分类、校区、价格区间）的组合查询。

## 2. 接口 `GET /api/item/search`

| 参数 | 类型 | 必填 | 说明 |
|------|------|:--:|------|
| keyword | string | 否 | ≤20字，FULLTEXT MATCH |
| type | int | 否 | 1=求购, 2=转让 |
| categoryId | int | 否 | |
| campus | string | 否 | |
| priceRange | string | 否 | below50 / 50to200 / above200 / negotiable |
| page | int | 否 | 默认1 |
| size | int | 否 | 默认10 |

## 3. 查询构建（Dynamic SQL）

`ItemMapper.ItemSqlProvider.searchItems()` 根据参数动态拼接：

- 基础池：`item_status=0 AND expire_at>NOW()`
- 关键词：`MATCH(content) AGAINST(#{keyword} IN BOOLEAN MODE)`（利用 FULLTEXT 索引）
- 结构化字段：逐个 AND 拼接
- 价格区间：below50(price<50) / 50to200(50≤price≤200) / above200(price>200) / negotiable(price IS NULL)
- 排序：有关键词→按相关度 DESC；无→按 created_at DESC

## 4. 价格区间枚举

| 参数值 | SQL |
|--------|-----|
| below50 | `price < 50` |
| 50to200 | `price >= 50 AND price <= 200` |
| above200 | `price > 200` |
| negotiable | `price IS NULL` |
