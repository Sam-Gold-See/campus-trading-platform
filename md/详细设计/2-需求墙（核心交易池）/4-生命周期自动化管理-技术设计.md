# 生命周期自动化管理 - 后端技术设计文档

## 1. 概述

`ItemExpireTask` 定时任务自动下架过期帖文。

## 2. 实现

```java
@Scheduled(cron = "0 0 * * * ?")  // 每小时执行
public void expireItems() {
    int count = itemMapper.expireItems();
    if (count > 0) log.info("过期帖文下架完成，处理 {} 条", count);
}
```

## 3. SQL

```sql
UPDATE item
SET item_status = 2
WHERE item_status = 0 AND expire_at <= NOW()
```

- 单条 SQL，无分批 LIMIT
- 条件 `item_status=0` 保证幂等性
